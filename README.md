## Introduction
gRPC-Rust-Polaris provides a series of components based on gRPC-Rust framework, developers can use gRPC-Rust-Polaris to build distributed gRPC-Rust applications.


## Install

Add the following to your `Cargo.toml`:
```toml
[dependencies]
polaris-grpc = "0.1"
```
